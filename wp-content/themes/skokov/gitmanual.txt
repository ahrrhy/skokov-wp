Вкратце про GIT (из записей + коррективы по ходу практического применения):
1) Cоздаём новый репозиторий на github.com (с именем HW-6 например).
  2) Создаём у себя на компе новую папку (HW-6), в неё добавляем пустой файл html и нужные нам папки (css, sass, images, fonts).
3) В этой папке правой кнопкой мыши вызываем Git Bush
4) В командной строке вводим следующие команды:
git init
git add .
git commit –m “add files”
git remote add origin адрес репозитория
(например https://github.com/имя/HW-6.git)
и далее вводим команду:
git push –u origin master
6) Создаём новую ветку, для чего в командной строке вводим:
git checkout –b hw6
7) Добавить какие-нибудь изменения в index.html (например, набросать структуру, подключить файл стилей), сохранить
8) Вводим в командную строку:
git add index.html
git commit –m “new commit”
git push origin hw6
9) Создаём ветку для хедера:
git checkout –b header
10) Добавляем в index.html наш header, сохраняем
11) В командной строке вводим:
git diff
(смотрим изменения в нашей ветке), далее вводим:
git add index.html
git commit –m “add header”
git push origin header
только это упрощённый пример, помните, что Олександр просил создавать одну общую папку для всех домашек сначала
